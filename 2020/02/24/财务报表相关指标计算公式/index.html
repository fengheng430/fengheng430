财务报表相关指标计算公式

发表于 2020-02-23
本文字数： 583 阅读时长 ≈ 1 分钟

资产= 负债+所有者权益

资产代表的是总资产，所有者权益就是净资产

  • 流动性比率

流动比率 = 流动资产/流动负债

速动比率 = （流动资产-存货）/流动负债

  • 当速动比率大于1时，若流动资产与流动负债同时减少相同数量，则速动比率提高，流动比率也提高。
  • 流动比率总是大于速动比率的，相对于流动比率来说，速动比率对于短期偿债能力的衡量更加直观可信。
  • 流动资产的内部转换不会改变流动比率，但可能改变速动比率。比如，企业用现金购买存货，流动比率不变，速动比率却会降低。
  • 财务杠杆比率

资产负债率 = 负债/资产

负债权益比 = 负债/所有者权益

权益乘数 = 资产/所有者权益

利息倍数 = EBIT（息税前利润）/利息费用

  • 营运效率指标

存货周转率 = 销售成本/年均存货

应收账款周转率 = 销售收入/年均应收账款

总资产周转率 = 销售收入/年均总资产

  • 盈利能力指标

销售利润率 = 净利润/销售收入

资产收益率 = 净利润/总资产

净资产收益率 = 净利润/所有者权益

净资产收益率高，说明企业利用其自有资本获利的能力强

  • 杜邦分析法

净资产收益率 = 净利润/所有者权益 = （净利润/总资产）*（总资产/所有者权益）

​ = 资产收益率*权益乘数

资产收益率 = 净利润/总资产 = （净利润/销售收入）*（销售收入/总资产）

​ = 销售利润率*总资产周转率

净资产收益率 = 销售利润率 * 总资产周转率 * 权益乘数

杜邦分析法利用主要的财务比率之间的关系来综合评价企业的财务状况，属于综合分析的方法。
